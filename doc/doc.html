Neregistrovaný/Registrovaný uživatel

Uživatel aplikace se může kdykoliv přihlásit, odhlásit, zobrazit profil nebo vstoupit do jemu povolené části administrace z horní lišty.

Objednání

Neregistrovaný uživatel má možnost si vybrat z pokrmů v nabídce a následně dokončit objednání registrací (vyplnění minimálních údajů + hesla pro přiští použití, dále se stává strávníkem). Postupně přidává do "košíku" jednotlivé pokrmy, které si chce objednat (stav se ukládá do cookies) a následně může objednávku dokončit a zaslat k zpracování. Z restaurace nelze objednat, pokud je zavřená a nebo již nepříjmá objednávky. Některé pokrmy mohou být označené jako nedostupné a nelze je přidat do objednávky. (facility_detail.html, views.py - facility_detail())

Prohlížení provozoven a jejich nabídek

Kdokoliv vidí všechny provozovny uložené v databázi a může nimi listovat, vybrat si danou provozovnu a následně je přesměrován do detailu provozovny, kde může položky pouze procházet a případně hned objednat. (index.html, views.py - index())

Instalace

Stručně popište:
  • Nainstalovat python v 3.7.4 a pip
  • Pomocí pip nainstalovat moduly uvedené v requirements.txt
  • Použít inicializační skript pro naplnění databáze a provedení počátečních migrací - db_init.sh (lokálně se vytvoří sqlite databáze, na heroku běží postgre)
  • Zapnout server pomocí python manage.py runserver
